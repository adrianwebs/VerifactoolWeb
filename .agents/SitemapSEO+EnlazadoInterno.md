# 🚫 Evitar canibalizaciones

## 1. Una keyword = una URL
- Incorrecto: duplicar páginas similares
- Correcto: una única URL fuerte

## 2. Diferenciar intención
- Informativa → blog
- Comercial → landing

## 3. Jerarquía clara
- Pilar → subpáginas → blog

## 4. Enlazado dirigido
- Priorizar:
  - /software-facturacion/
  - /verifactu/
  - /precios/
